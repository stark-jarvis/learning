# WebPack 相关
> 本质上，`webpack` 是一个现代 JavaScript 应用程序的静态模块打包器`(module
> bundler)`。当`webpack`处理应用程序时，它会递归地构建一个依赖关系，其中包含应用程序需要的每个模块，然后将所有这些模块打包成一个或多个`bundle`。

## 四个核心概念：
* 入口（entry）
* 输出（output）
* loader
* 插件（plugins）
